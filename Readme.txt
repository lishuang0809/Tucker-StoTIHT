Requirements:
Tensorlab (https://www.tensorlab.net/)

This package is a collection of code used in the paper "Stochastic Iterative Hard Thresholding for Low-Tucker-Rank Tensor Recovery". The code in this package is sufficient to reproduce all of the results in this paper.

The code has been tested to run in MATLAB R2020a.

If you use this codes in an academic paper, please cite our paper:
R. Grotheer, S. Li, A. Ma, D. Needell, and J. Qin, Stochastic Iterative Hard Thresholding for Low-Tucker-Rank Tensor Recovery, Proc. Information Theory and Applications, La Jolla, California, February 2020.
